# Homebridge Octoprint

## Setup

`npm install -g Cosmo/homebridge-octoprint`

## Config

Under accessories, add:

```javascript
{
  "accessory": "Octoprint",
  "name": "Octoprint",
  "api_key": "[YOUR_API_KEY]",
  "server": "[OCTOPRINT_ADDRESS]"
}
```
